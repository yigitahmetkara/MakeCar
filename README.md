Arabayapmak - Java OOP Uygulaması
Bu proje, Java programlama dilinde Nesne Yönelimli Programlama (OOP) prensiplerini temel alarak oluşturulmuş bir araç simülasyonu çalışmasıdır. Proje kapsamında sınıflar, nesne oluşturma ve metot kullanımı gibi temel programlama kavramları uygulanmıştır.

Özellikler
Nesne Modelleme: Gerçek dünya objesi olan "Araba"nın kod ortamında modellenmesi.

Özellik Yönetimi: Araçların marka, model, renk ve motor gücü gibi niteliklerinin tanımlanması.

Davranış Tanımlama: Araçların hızlanma, durma ve diğer fonksiyonel özelliklerinin metotlar aracılığıyla simüle edilmesi.

Teknik Detaylar
Dil: Java

Geliştirme Ortamı: Eclipse IDE

Konsept: Nesne Yönelimli Programlama (Encapsulation, Classes, Objects)

Kurulum ve Kullanım
Projeyi bilgisayarınıza indirin veya klonlayın:

Bash
git clone https://github.com/yigitahmetkara/arabayapmak.git
Eclipse IDE'sini açın.

File > Import > General > Existing Projects into Workspace adımlarını izleyerek projeyi dahil edin.

Main sınıfını çalıştırarak araç nesnelerinin oluşturulma ve çalışma süreçlerini konsoldan izleyin.
